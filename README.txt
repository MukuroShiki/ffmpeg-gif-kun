# FFmpeg GUI Kun - 動画エンコード・GIF変換ツール

## 概要
FFmpeg GUI Kunは、FFmpegを使用した動画エンコードとGIF変換を簡単に行えるGUIアプリケーションです。
直感的なタブ式インターフェースで、複雑なFFmpegコマンドを知らなくても高品質な動画変換が可能です。

**🎯 特徴**
- **簡単操作**: ドラッグ&ドロップでファイル選択
- **FFmpeg不要**: 自動ダウンロード機能付き（初回起動時）
- **Python不要**: スタンドアロン実行ファイルで提供
- **日本語対応**: 完全日本語インターフェース
- **リアルタイム**: 進行状況をリアルタイム表示

## 主な機能

### 動画エンコード
- 様々な出力フォーマット対応 (MP4, AVI, MOV, MKV, WebM等)
- 豊富なコーデック選択 (H.264, H.265, VP8, VP9等)
- 解像度・FPS・品質の自由な設定
- CRF/ビットレート両方の品質設定対応
- エンコードプリセットによる速度・品質調整
- カスタムプリセット保存機能

### GIF変換
- 動画からGIFアニメーション作成
- 解像度・FPS調整可能
- 品質レベル選択 (低・標準・高)
- 時間範囲指定対応
- プレビュー機能付き

### その他の機能
- ドラッグ&ドロップ対応 (ファイル選択)
- リアルタイム進行状況表示
- 処理中断機能
- ファイル情報表示
- 処理中のウィンドウ終了警告
- 設定の自動保存・読み込み

## システム要件
- **Windows**: Windows 10以上推奨
- **メモリ**: 2GB以上推奨
- **ディスク容量**: 200MB以上（FFmpeg含む）

## 簡単インストール & 起動方法

### ⚡ 今すぐ使用（推奨）
1. **`FFmpeg-GUI-Kun.exe` をダブルクリック**
2. **完了！** - 自動でFFmpegがダウンロードされます

### 📦 配布パッケージ版
1. **`FFmpeg-GUI-Kun-windows.zip` を解凍**
2. **`FFmpeg-GUI-Kun.exe` を実行**
3. **完了！**

## 初回起動について
初回起動時に以下が自動実行されます：
- FFmpegバイナリの自動ダウンロード
- 設定フォルダの作成
- 必要なライブラリの初期化

**💡 ヒント**: 初回起動は少し時間がかかる場合がありますが、2回目以降は高速に起動します。

## 使用方法

### 🚀 基本操作
1. **アプリケーションを起動** - `FFmpeg-GUI-Kun.exe` をダブルクリック
2. **タブを選択** - 「動画エンコード」または「GIF変換」
3. **ファイルを選択** - 参照ボタンまたはドラッグ&ドロップ
4. **設定を調整** - 品質・解像度など
5. **開始ボタンで実行** - リアルタイムで進行状況確認

### 📹 動画エンコード手順
1. **出力フォーマット選択** - MP4, AVI, MOV等から選択
2. **コーデック設定** - H.264, H.265等のビデオコーデック
3. **品質設定** - CRF値（推奨：18-28）またはビットレート
4. **解像度・FPS** - 必要に応じて調整
5. **プリセット** - 速度重視・品質重視・バランス型から選択

### 🎨 GIF変換手順
1. **基本設定** - 解像度（幅×高さ）とFPS
2. **品質選択** - 低品質（軽量）・標準・高品質（重い）
3. **時間範囲** - 開始時間と終了時間を指定可能
4. **プレビュー** - 変換前に結果をプレビュー確認

## ⌨️ キーボードショートカット
- **Ctrl+O**: ファイルを開く
- **Ctrl+S**: 設定を保存
- **Ctrl+Q**: アプリケーション終了
- **F1**: ヘルプ表示
- **Esc**: 処理中断

## 🛠️ トラブルシューティング

### ❓ よくある質問

**Q: アプリが起動しない**
A: Windows Defenderで実行がブロックされている可能性があります。
   「詳細情報」→「実行」をクリックしてください。

**Q: FFmpegダウンロードに失敗する**
A: インターネット接続を確認し、ファイアウォール設定をご確認ください。

**Q: 変換が途中で止まる**
A: 入力ファイルが破損している可能性があります。別のファイルでお試しください。

**Q: 変換が遅い**
A: プリセットを「速度重視」に変更するか、解像度を下げてお試しください。

### 🔧 エラー対処法

**「FFmpegが見つかりません」エラー**
- アプリを再起動してFFmpegの自動ダウンロードを実行
- 手動でFFmpegをインストール後、アプリを再起動

**「ファイルにアクセスできません」エラー**
- ファイルが他のアプリで使用中でないか確認
- 十分なディスク容量があるか確認
- 管理者権限でアプリを実行

## 💡 パフォーマンスのヒント

### 🚀 高速変換のコツ
- **プリセット選択**: 「速度重視」を選択
- **解像度調整**: 必要以上に高解像度にしない
- **CRF値**: 23-28程度で十分な品質
- **ハードウェア**: SSDを使用すると高速

### 📏 品質vs容量のバランス
- **高品質**: CRF 18-23 (大きなファイルサイズ)
- **標準品質**: CRF 23-28 (バランス良好)
- **軽量**: CRF 28-35 (小さなファイルサイズ)

## 📋 技術仕様
- **GUI**: tkinter（Python標準ライブラリ）
- **動画処理**: FFmpeg 4.4+
- **設定管理**: JSON形式（自動保存）
- **ログ出力**: アプリケーションログ対応
- **対応OS**: Windows 10+（メイン対応）
